1. In todays Code Club we are making a 3D maze game using UPBGE, the first few steps of this guide will show you how to install UPBGE
  2. The best way to run UPBGE is to clone the code using command prompt, you can also download if from the website but this is slower. To download using command prompt press win + R and type cmd, then run the command curl -k -L -o upbge.7z https://github.com/UPBGE/upbge/releases/download/v0.36.1/upbge-0.36.1-windows-x86_64.7z
  3. Or you can visit https://upbge.org/#/download and press download
  4. Unzip the file by opening file explorer to the download location, if you used command prompt this will be your user folder, and then right clik on the 7z archive and press extract all
  5. Navigate to the folder with your extracted files, then open the only folder that is inside of it, launch blender.exe
  6. Visit https://github.com/code-club99/maze-game/blob/main/maze_game.blend and press the download icon
  7. In UPBGE press file then open and select the file you just downloaded
  8. Press the 0 key on your num pad, or press view -> cameras -> active camera
  9. Open the logic bricks editor by pressing the clock icon in the bottom left then pressing logic bricks editor under scripting
  10. Select the object Cube under scene collection in the top left
  11. Press add sensor then press keyboard
  12. Press the blank space next to key, then press the up arrow or W
  13. Press add actuator then press motion
  14. In the motion block set the Y under loc to -0.10, UPBGE uses XYZ for it's positions
  15. Join up the sensor and the actuator by dragging the link icon to each other
  16. Repeat theese steps for bacwards using 0.10 for the Y instead of -0.10
  17. For left and right we will use a similar procces but instead of using loc we will use rot to rotate intead of move, for left set Z to -1 and for right Z to 1
  18. Press the camera icon on the right then press standalone start, then try moving forwards, backwards, left, and right

  20. Go into the mateirals editor, it is in the left hand menu and looks like a ball
  21. Press new then pres the white next to base color
  22. Select a colour
  23. Select the maze and repeat the steps to add a material to it
  24. Select the character(the cube)

  26. Press M/P
  27. Select next level
  28. Press add actuator then scene
  29. Set the mode to set scene and select Scene.001
